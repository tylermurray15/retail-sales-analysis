# retail-sales-analysis

SQL, Excel, and Tableau project analyzing retail sales trends, profitability, customer segments, and regional performance.

# Project Overview

This project analyzes retail sales data to identify sales trends, profitable products, underperforming categories, customer purchasing patterns, and regional performance.

# Tools

- Excel
- SQL
- Tableau

# Business Questions

1. Which products generate the most revenue?
2. Which product categories generate the most revenue?
3. Which products generate the least revenue?
4. Which months have the highest sales?
5. Which regions perform best?
6. Which customer segments spend the most?
7. Are there products with strong sales but weak profits?
8. How does performance change over time?
9. Which categories should the business prioritize?
10. Where might the business be losing money?
